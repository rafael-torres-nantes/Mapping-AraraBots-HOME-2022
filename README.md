## Informações do GIT

### Instalando o GIT

```bash
$ sudo apt-get install git
```

```bash
$ sudo add-apt-repository ppa:git-core/ppa
```


### Iniciar o Repositório GIT

```bash
$ git init
```

### Salvar os Arquivos no GIT

```bash
$ git status
```
```bash
$ git pull
```
```bash
$ git add .
```
```bash
$ git commit -m "Nome do Commit e das Alterações"
```
```bash
$ git push --force origin master
```
